# TODOアプリ 仕様書

## 概要

個人利用向けのシンプルなTODOアプリケーション。タスクの管理と期限設定をサポートし、データはローカルストレージに保存する。

## 機能要件

### タスク管理

| 機能 | 説明 |
|------|------|
| タスク追加 | タイトルと任意の期限日を入力して追加 |
| タスク編集 | タイトルと期限日をインライン編集 |
| タスク削除 | 確認ダイアログ付きで削除 |
| 完了切り替え | チェックボックスで完了/未完了を切り替え |
| フィルタリング | 完了済みタスクの表示/非表示 |

### データ永続化

- ローカルストレージへの自動保存
- ページリロード後もデータを維持
- 旧バージョンデータのマイグレーション対応

## データ構造

```typescript
interface Task {
  id: string;        // UUID
  title: string;     // タスク名
  completed: boolean; // 完了フラグ
  dueDate?: string;  // 期限日（ISO形式、任意）
  createdAt: string; // 作成日時（ISO形式）
  updatedAt: string; // 更新日時（ISO形式）
}
```

## UI仕様

### 期限の色分け

- **期限切れ**: 赤色 (`text-red-500`)
- **本日期限**: オレンジ色 (`text-orange-500`)
- **期限内**: グレー (`text-gray-500`)

### 完了済みタスク

- 取り消し線 (`line-through`)
- 透過 (`opacity-50`)

### レスポンシブデザイン

- モバイル（スマートフォン）対応
- タブレット・デスクトップ対応

## 除外機能（スコープ外）

- ユーザー認証
- クラウド同期
- カテゴリ・タグ分け
- 優先度設定
- 通知機能
- チーム共有
